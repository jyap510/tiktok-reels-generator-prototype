# Prebuilt Profile Templates

Prebuilt profile templates provide recommended browser profile configurations for common platform workflows.

These templates demonstrate typical settings used for different platforms, including fingerprint configuration, proxy type, and regional settings. They can be used as starting points when creating new browser profiles in Gologin.

Templates are examples rather than fixed presets and can be adjusted depending on your workflow.

## TikTok Profile Template

Recommended configuration for TikTok account environments.

| Parameter           | Recommended setting     |
| :------------------ | :---------------------- |
| Browser type        | Mobile-like environment |
| Proxy type          | Residential or mobile   |
| Proxy location      | Fixed region            |
| Fingerprint changes | Minimal                 |
| Timezone            | Match proxy location    |

TikTok relies heavily on device consistency and session behavior. Stable environments help reduce verification checks.

## Facebook / Meta Ads Profile Template

Recommended configuration for Facebook accounts and Meta Ads workflows.

| Parameter           | Recommended setting    |
| :------------------ | :--------------------- |
| Browser type        | Desktop                |
| Proxy type          | Residential            |
| Proxy location      | Fixed                  |

Facebook evaluates device signals and login history, so consistent environments are important.
