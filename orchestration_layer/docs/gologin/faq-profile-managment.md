# FAQ

## Can I run multiple profiles at the same time?

Yes. Gologin allows you to run multiple browser profiles simultaneously, depending on your system resources and subscription plan.

Each profile runs in an isolated browser environment with its own fingerprint, cookies, and proxy settings.

## Can I use the same proxy for multiple profiles?

Technically yes, but it is not recommended.

Using the same proxy for multiple profiles may link accounts together and reduce their trust score on certain platforms.

For best results, use **a unique proxy for each profile**.

## What happens if I close a profile incorrectly?

If a profile is closed before synchronization is completed, the latest session data may not be saved.

This can result in:

* lost cookies
* lost session state
* websites logging you out

Always close profiles normally and wait until the profile state returns to **Ready**.

## Can I launch the same profile on multiple devices?

This is technically possible but not recommended.

Running the same profile simultaneously on different devices may cause synchronization conflicts and increase the risk of account issues.

## Can I copy or duplicate a profile?

Yes. Goдogin allows you to **clone profiles**.

Cloning a profile copies its settings, fingerprint configuration, and environment, allowing you to quickly create similar profiles.

## Can I recover a deleted profile?

Yes. Deleted profiles can be restored for a limited time from the **Restore profiles** section in Settings.

If the profile has not been permanently removed, it can be recovered with its data intact.

## Why did my account get logged out after launching a profile?

This can happen if:

* the proxy location changed
* the fingerprint changed significantly
* cookies were invalid or expired

To reduce this risk, keep proxy, fingerprint, and timezone settings consistent.
