# GoLogin Web, Mobile & Chrome OS

> How to use GoLogin through the web app, Android app, and on Chrome OS devices

## GoLogin Web (app.gologin.com)

You can access all your profiles from any browser at **app.gologin.com** without installing the desktop app. This is useful for:

* Chrome OS devices
* Shared or restricted computers
* Quick access from any device

**Cloud session limits apply:**

* Professional: 1 concurrent cloud session
* Business: 2 concurrent cloud sessions
* Enterprise: 3 concurrent cloud sessions

All profile data is synced between desktop and web versions.

## GoLogin Android App

The GoLogin Android app allows you to **view and manage** your profiles from your phone or tablet. However, it does **not** support launching browser profiles directly.

What you CAN do:

* View your profile list
* Check profile status
* Manage account settings

What you CANNOT do:

* Launch profiles
* Browse inside profiles
* Run automations

<Info>
  "Cloud Android" (Android anti-detect environments) is a separate feature from the GoLogin Android app. Cloud Android lets you run isolated Android environments in the cloud -- it does not require the mobile app.
</Info>

## Chrome OS

GoLogin works on Chrome OS through the web version (app.gologin.com). No installation needed -- open your browser and log in. Cloud session limits apply based on your plan.
