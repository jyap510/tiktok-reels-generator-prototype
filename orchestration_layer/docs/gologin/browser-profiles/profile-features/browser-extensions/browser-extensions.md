# Browser extensions overview

Extensions inside a Gologin profile allow you to customize each browser environment individually. Since every profile in Gologin works as a separate browser with its own fingerprint and storage, extensions are installed and managed **per profile**, not globally. This gives you full control over which tools run inside each account environment.

## What are profile extensions?

**Profile extensions** are **Chrome-compatible** browser extensions that you install **directly inside a specific Gologin profile**.

Each profile:

* Has its own extension list
* Stores extension data separately
* Keeps extension settings isolated
* Does not share extensions with other profiles

This means you can run different setups for different accounts without overlap.

## Why use extensions in Gologin profiles?

Extensions are often essential for:

* Automation tools
* Cookie management
* Ad verification
* Web scraping assistance
* Productivity workflows

Because profiles are isolated, you can safely use different tools across accounts without cross-contamination of data.

## Extensions and Fingerprint stability

Extensions can affect browser behavior.

Some extensions:

* Modify request headers
* Inject scripts
* Interact with WebRTC
* Change canvas behavior

If you're working with sensitive platforms (ads, e-commerce, crypto), test new extensions carefully before using them on warmed-up accounts.

## Best practices

* Avoid installing unnecessary extensions
* Use consistent extensions per account type
* Don't frequently add/remove extensions on established profiles Abrupt changes in extension setup may look suspicious on high-security platforms.
