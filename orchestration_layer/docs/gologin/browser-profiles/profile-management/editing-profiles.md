# Editing profiles

## Changes from the profiles page

You can change some settings right from the main **All profiles** page. You can manage such settings as: **name, proxies, notes, status, tags**.

## Quick profile settings

To quickly make some changes to your profiles, just click the **Settings** button on the profile. You then can define new settings in the sidebar.

<Frame>
    <img src="https://mintcdn.com/gologin/pCb28zkpcbLZzMmX/images/quick_profile_settings.gif?s=1ba980e5d93d8c4517c7915c4126515a" alt="Quick Profile Settings" width="1391" height="793" data-path="images/quick_profile_settings.gif" />
</Frame>

## Main profile settings

You can access the main profile settings by **clicking 3 dots on the profile > Settings**. Here you can get more parameters and a nuanced description for each of them.
