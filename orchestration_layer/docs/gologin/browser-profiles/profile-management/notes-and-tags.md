# Notes, Tags and Status

These text profile fields help users to differentiate, filter and find profiles with ease.

## Notes

**Notes** in Gologin allow you to store additional information directly within each browser profile. This is useful for keeping track of account details, reminders, or any context related to a specific profile.

To **add a note to the profile**, simply click on the notes of the profile and start typing.

    <img src="https://mintcdn.com/gologin/WOwT3qt1z8aRNb0n/images/add_a_note.gif?s=40cf630b928c36c6a95d1dd74f03ebc2" alt="Add A Note" width="1391" height="793" data-path="images/add_a_note.gif" />

Notes support basic **formatting**, including **bold**, *italic*, ~~striketrough~~, background color, <u>underline</u>. \
Select the text and apply it!

## Tags and Status

**Tags** allow you to label profiles with keywords to help categorize and filter them by account type, platform or project, while **Status** displays the workflow state.

<Info>
  These fields are hidden by default. Visit this page to learn how to enable them in the settings.
</Info>

<Tip>
  The key difference is that a profile can have multiple tags, but only one status.
</Tip>

### Create or add

To **create a new tag or status**, click on the respective field in the profile and start typing. You can also choose **a color** while creating or editing it.

You can **assign existing tags** to a profile by selecting them from the list.

    <img src="https://mintcdn.com/gologin/WOwT3qt1z8aRNb0n/images/create_and_add_tags_and_status-1.gif?s=ed4261fadf22320e258fab6cf550b928" alt="Create And Add Tags And Status" width="1391" height="793" data-path="images/create_and_add_tags_and_status-1.gif" />

### Filter by

You can filter profiles by status or tags. To do this, click on the corresponding field and select the tag or status you want to filter by.

To clear the filter, click the arrow next to the field category.

<Frame>
    <img src="https://mintcdn.com/gologin/WOwT3qt1z8aRNb0n/images/filter_by_status_or_tags.gif?s=3212bd4a7f41f04f426847ba0995ae66" alt="Filter By Status Or Tags" width="1391" height="793" data-path="images/filter_by_status_or_tags.gif" />
</Frame>

### Modify tags and status

* To **remove** a tag or status from a profile:\
  Click the field → click **X** next to the item
* To **delete** a tag or status entirely:\
  Click the field → click the **three dots** next to the item in the list → **Delete**

You can also change tag and status colors from this list.

## Teamwork

These fields behave differently when working with shared profiles, folders, and workspaces.

* **Notes are fully synced**. Any changes made to notes will be visible to all team members.
* **Tags and Status are user-specific**. When you share a profile, recipients will not see your tags or status and can assign their own based on their workflow.

<Info>
  Team members need "can edit" access to change profile notes, status, and tags
</Info>
