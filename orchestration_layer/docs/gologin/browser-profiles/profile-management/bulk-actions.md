# Bulk actions

Bulk actions in Gologin allow you to apply the same action to multiple profiles at once.

To use bulk actions, first select several profiles in the list using the checkboxes. After that, the bulk action panel appears at the top of the page.

## How to use bulk actions

1. Select multiple profiles using checkboxes
2. The bulk action panel will appear at the top
3. Choose the action you want to apply

## When to use bulk actions

Bulk actions are most useful when you need to:

* Launch multiple profiles for daily work or warm-up
* Update settings in batches
* Organize profiles across folders and projects
* Manage access for team members
* Maintain or refresh fingerprints at scale

## What you can do with bulk actions

### **Run profiles**

Select the profiles and click **Run profile** to launch all selected profiles.

Use this to start multiple accounts for daily work or warm-up sessions.

<Note>
  Running too many profiles at once may affect performance depending on your system.
</Note>

Select the profiles, click **Proxy**, and add or update proxy settings for all selected profiles.
